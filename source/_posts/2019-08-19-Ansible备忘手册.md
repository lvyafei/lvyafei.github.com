## 1.生成秘钥

ssh-keygen -t rsa

## 2.进入目录 

```shell
cd ~/.ssh/
```
<!--more-->

## 3.本机加入授权

```shell
cat id_rsa.pub >> authorized_keys
```

## 4.修改文件权限

```shell
chmod 600 ./authorized_keys
```

## 5.把第一个节点上的 authorized_keys 发送到第二个节点上

```shell
scp authorized_keys node2.cn:~/.ssh/
```

## 6.测试 ssh 免密登录

第一个节点：
ssh node2.cn 'ls'
