## 0.前言

Redis运维手册

## 1.连接数优化

使用密码登录Redis

```
redis-cli -a 123456
```

查看配置的最大连接数：

```
CONFIG GET maxclients
//设置最大连接数：
CONFIG set maxclients 10
```

查看当前连接数：

```
info clients
```

查看连接的客户端详情：

```
CLIENT LIST
```
